## Technologies
- HTML5, CSS3, JavaScript (ES6+)
- Local Storage API

## 🎯 Features

### 1. 📊 Counter App
- Increment, decrement, and reset counter
- Keyboard shortcuts: ↑ (increment), ↓ (decrement), R (reset)
- Demonstrates Scope, Closures, and Hoisting

### 2. ✅ To-Do List
- Add, delete, and mark tasks complete
- Real-time statistics (total/completed)
- Keyboard support (Enter to add)
- Demonstrates DOM Manipulation and Events

### 3. 💬 Quote Generator
- Random quotes from curated collection
- Copy quote to clipboard
- Add/remove favorites (saved in browser)
- Keyboard shortcuts: Space (new), C (copy), F (favorite)
- Demonstrates Promises, Async/Await, and Local Storage
