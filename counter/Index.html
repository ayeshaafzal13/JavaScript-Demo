🧮 Counter App

Demonstrates: Scope • Closures • Hoisting

0
🧠 Concepts Demonstrated:

Scope: The count variable is inside createCounter() function. It cannot be accessed from outside.

Closure: Inner functions (increment, decrement, reset) "remember" the count variable even after the outer function returns.

Hoisting: The function createCounter() is called before it's defined in the code. This works because of hoisting!
